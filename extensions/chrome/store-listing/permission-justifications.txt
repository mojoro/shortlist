PERMISSION JUSTIFICATIONS — Paste these into the Chrome Web Store form
======================================================================

The Chrome Web Store asks you to justify each permission. Copy/paste these:

activeTab:
"The extension reads the current page's HTML content when the user clicks the extension icon, to extract job listing details (title, company, description, etc.) for import into their Shortlist account. Content is only read on explicit user action — never in the background."

scripting:
"Used to inject a lightweight content extraction script into the active tab when the user clicks the extension icon. This is necessary because the content script may not be pre-loaded on pages. The script only reads page content — it does not modify the page."

storage:
"Stores the user's recent import history (last 5 imports) and preferences locally in the browser. No sensitive data is stored — authentication relies on the user's existing browser session cookie."

Host permission — https://shortlist.johnmoorman.com/*:
"The extension communicates with the Shortlist API to authenticate the user, fetch their search profiles, extract job details using AI, and save imported job listings. All API calls use the user's existing session cookie for authentication."

content_scripts — <all_urls>:
"The content script is injected on all pages so that job listings can be extracted from any job board the user visits (LinkedIn, Indeed, Greenhouse, corporate careers pages, etc.). The script is lightweight (under 1KB) and only performs work when the user explicitly triggers an extraction by clicking the extension icon."
