CHROME WEB STORE SUBMISSION CHECKLIST
=====================================

This folder contains everything you need to submit the Shortlist Job Importer
extension to the Chrome Web Store.

Files in this folder:
  shortlist-extension-v1.0.0.zip  — The extension package to upload
  store-description.txt           — Name, description, category (copy/paste)
  permission-justifications.txt   — Justification text for each permission
  privacy-policy.html             — Host this at shortlist.johnmoorman.com/privacy-extension
  icon128-store.png               — 128x128 icon for the store listing

Steps:
  1. Host the privacy policy
     Upload privacy-policy.html to your site (e.g., shortlist.johnmoorman.com/privacy-extension)
     You'll need the URL for the store listing form.

  2. Take a screenshot (required)
     The Chrome Web Store requires at least one 1280x800 or 640x400 screenshot.
     Open a job page (e.g., Greenhouse), click the extension, and take a screenshot
     showing the popup with extracted job details. Crop to 1280x800.

  3. Go to https://chrome.google.com/webstore/devconsole
     Click "New Item" and upload shortlist-extension-v1.0.0.zip

  4. Fill in the store listing
     Copy text from store-description.txt for name, description, category.

  5. Fill in the privacy tab
     Paste the hosted privacy policy URL.
     Select "This extension does not collect or transmit user data" for the
     single purpose description, then describe: "Extracts job listing content
     from the current page and sends it to the user's Shortlist account for import."

  6. Fill in the permissions justifications
     Copy text from permission-justifications.txt.

  7. Upload the screenshot(s)

  8. Submit for review (typically 1-3 business days)
